 
 CINS实验室_SCCA项目后端代码 
================================================================================

利用`spring` + `Mybaits`框架实现

* `Mybatis`实现`DAO`层访问
* `spring`实现`DAO`层、`Service`层的注入
* `Spring-mvc`实现`Controller`的注入
* `org.springframework.test.web.servlet.MockMvc`实现`Controller`测试

> 接口说明在`src/main/webapp/index.html`中
